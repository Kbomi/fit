# 체질노트

> 사상체질 경향을 가볍게 확인하고, 내 몸의 컨디션 흐름을 생활 루틴으로 이어보는 웹 서비스

## 소개

체질노트는 태양인, 소양인, 태음인, 소음인으로 나뉘는 사상체질 경향을 테스트하고 결과를 리포트처럼 보여주는 모바일 중심 웹앱입니다.

사용자는 짧은 질문에 답한 뒤 체질 비율, 성향 요약, 음식 가이드, 생활 루틴, 체형 참고 리포트를 한 화면에서 확인할 수 있습니다. 결과는 확정 진단이 아니라 일상에서 몸의 리듬을 살피기 위한 참고 정보로 제공됩니다.

## 주요 기능

- 10개 문항 기반 사상체질 경향 테스트
- 태양인, 소양인, 태음인, 소음인 점수 계산과 비율 표시
- 체질별 성향, 컨디션 포커스, 음식 추천, 주의 음식 안내
- 아침, 점심, 저녁으로 나뉜 생활 루틴 제안
- 남성/여성 체형 예시 이미지가 포함된 체형 경향 리포트
- Web Share API와 클립보드 복사를 활용한 결과 공유
- Node.js 기본 테스트 러너 기반 점수 계산 로직 테스트

## 화면 구성

```text
홈/히어로
  서비스 소개, 테스트 시작, 리포트 미리보기

체질 체크
  진행률, 질문 카드, 이전/초기화 컨트롤

결과 리포트
  체질 비율, 컨디션 포커스, 루틴, 음식, 공유

체형 가이드
  체질별 남성/여성 예시 이미지와 설명
```

## 기술 스택

- HTML, CSS, JavaScript
- ES Modules
- Vite 개발 서버
- Node.js `node:test`

## 시작하기

```bash
npm install
npm run serve
```

개발 서버가 실행되면 터미널에 표시되는 로컬 주소로 접속합니다.

## 테스트

```bash
npm test
```

점수 합산, 동점 처리, 미응답 문항 처리 로직을 확인합니다.

## 프로젝트 구조

```text
assets/                 체질별 이미지와 일러스트

src/main.js             화면 렌더링과 사용자 인터랙션

```

## 안내

체질노트는 의료 진단, 치료, 예방 목적의 서비스가 아닙니다. 결과는 생활 참고용 경향 안내로만 활용해 주세요. 건강 문제나 지속적인 불편함이 있다면 한의사, 의사 등 전문가와 상담하는 것을 권장합니다.
