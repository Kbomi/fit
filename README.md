# 체질노트

사상체질 테스트를 통해 사용자의 체질 경향을 확인하고, 성격, 음식, 체형 가이드를 제공합니다.

## 주요 기능

- 태양인, 소양인, 태음인, 소음인 경향 테스트
- 응답 기반 점수 계산 및 비율 표시
- 체질별 생활 참고 가이드 제공
- 성별 선택에 따른 체형 일러스트 표시

## 실행 방법

```bash
npm run serve
```

## 테스트

```bash
npm test
```

## 프로젝트 구조

```text
assets/                 체질별 SVG 이미지
docs/                   구현 메모
src/                    앱 콘텐츠, 화면 렌더링, 점수 계산 로직
tests/                  Node.js 테스트
service-plan.md         서비스 기획서
```

## 안내

이 서비스는 의료 진단이 아닌 생활 참고용입니다. 정확한 진단이나 치료 판단은 전문가 상담이 필요합니다.
